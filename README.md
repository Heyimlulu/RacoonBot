# racoon-bot

A multi function discord bot.

## Getting started

### Prerequisites

You need to install **Node.js** on your computer to get this bot running

### Installing

```
Clone this repository
cd to the bot directory
```

To run the bot

```
node index.js
```

Don't forget to setup your bot token before running it
